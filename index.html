Análisis Clínico de Microdeleción 15q11.2

BP1-BP2 - Alteración genética vinculada a condiciones del neurodesarrollo

Sobre este estudio

Este sitio presenta una síntesis interactiva sobre la microdeleción 15q11.2 (BP1-BP2), una alteración genética vinculada a condiciones del neurodesarrollo como el autismo, la hipotonía y retrasos cognitivos. El objetivo es facilitar el análisis clínico y la recolección de datos mediante un formulario accesible para médicos, terapeutas y familiares.

Puedes contribuir al estudio agregando un nuevo caso clínico mediante el formulario. Los datos serán revisados y utilizados para enriquecer futuras versiones de esta plataforma.
